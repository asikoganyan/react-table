## Clone and start the project
### `git clone git@github.com:asikoganyan/react-table.git`
To clone the project from repository.
### `cd react-table`
Move into the project directory.
### `npm install`
To install npm packages specified in `package.json`.
### `npm start`
Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
